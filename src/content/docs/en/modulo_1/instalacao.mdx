## Installation

#### Main Application:

The installation of the application will depend on your operating system. Access the [official Zotero download page](https://www.zotero.org/download/) to download the version compatible with your computer.
<Tabs>
  <TabItem icon="laptop" label="Windows">
    Download the `.exe` installer from the official website, double-click the downloaded file, and follow the standard installation wizard.
  </TabItem>
  
  <TabItem icon="apple" label="macOS">
    Open the downloaded `.dmg` file and drag the Zotero icon to your **Applications** folder. 
    
    :::tip[Cleanup Tip]
    After installation, you can eject the installer from your system and delete the `.dmg` file to free up space.
    :::
  </TabItem>
  <TabItem icon="linux" label="Linux (Debian/Ubuntu)">
      #### Installation via Flatpak
  
      The traditional package for Linux can be downloaded directly from the [official Zotero page](https://www.zotero.org/download/). 
      
      However, if you prefer to manage your applications in another way, you can install the **Flatpak** version. If you are using Debian or Ubuntu and do not have Flatpak configured yet, follow the steps below in your terminal:
  
      :::tip[Using another distribution?]
      The commands below are for Debian/Ubuntu-based distributions. If you use Fedora, Arch, or others, check how to install on your system at the [official Flatpak website](https://flatpak.org/setup/) and access [Flathub](https://flathub.org/pt-BR). Additionally, it is worth checking if Zotero is available in your distribution's app store.
      :::
  
      ```bash
      # 1. Install Flatpak support on your system
      sudo apt install flatpak
  
      # 2. Add Flathub (the main Flatpak app repository)
      flatpak remote-add --if-not-exists flathub [https://dl.flathub.org/repo/flathub.flatpakrepo](https://dl.flathub.org/repo/flathub.flatpakrepo)
      
      # 3. Install Zotero
      flatpak install flathub org.zotero.Zotero
      ```
    </TabItem>
 </Tabs>

#### Browser Extension (Zotero Connector)

Simultaneously, we will need the browser extension installed for the course progression. This extension will help us send articles, web pages, etc., to Zotero.

<Tabs>
  <TabItem label="Chrome / Edge / Brave / Firefox" icon="window">
    Access [the official download page](https://www.zotero.org/download/); on the right side, next to the application download, there is the option to install the Zotero Connector.
  </TabItem>
  <TabItem label="Safari" icon="apple">
    The Zotero Connector for Safari comes built-in with the main Zotero application for macOS. 
    To enable it, open Safari, go to **Settings > Extensions**, and check the Zotero box.
  </TabItem>
</Tabs>

## Configuration

With the application and extension installed, let's configure Zotero:

### 1. Adjusting the Language

If the application is not in your desired language, follow these steps:

<Steps>
1. Open Zotero and access Preferences:
   * **On Windows/Linux:** Click the `Edit` menu > `Settings`.
   * **On macOS:** Click the `Zotero` menu (next to the Apple icon) > `Settings`.
2. Go to the **General** tab.
3. In the Language section, select **Português (Brasil)** (or your preferred language).
4. Click **OK** and restart the application to apply the changes.
</Steps>

---

### 2. Adding the ABNT Style

Zotero comes with several international formats (such as APA and Vancouver), but we need to add ABNT standards for our work.

<Steps>
1. Return to the **Preferences** menu and click the **Cite** tab.
2. Click the link **Get additional styles...**.
3. A Style Repository window will open. In the search bar, Search which styles you prefer.
4. Click the desired style to automatically install it into your list.
</Steps>
