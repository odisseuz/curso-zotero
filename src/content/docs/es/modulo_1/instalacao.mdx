## Instalación

#### Aplicación principal:

La instalación de la aplicación dependerá de tu sistema operativo. Accede a la [página oficial de descarga de Zotero](https://www.zotero.org/download/) para descargar la versión compatible con tu computadora.
<Tabs>
  <TabItem icon="laptop" label="Windows">
    Descarga el instalador `.exe` desde el sitio web oficial, haz doble clic en el archivo descargado y sigue el asistente de instalación estándar.
  </TabItem>

  <TabItem icon="apple" label="macOS">
    Abre el archivo `.dmg` descargado y arrastra el ícono de Zotero a la carpeta **Applications** (Aplicaciones).

    :::tip[Consejo de limpieza]
    Después de la instalación, puedes expulsar el instalador del sistema y eliminar el archivo `.dmg` para liberar espacio.
    :::
  </TabItem>
  <TabItem icon="linux" label="Linux (Debian/Ubuntu)">
      #### Instalación mediante Flatpak

      El paquete tradicional para Linux puede descargarse directamente desde la [página oficial de Zotero](https://www.zotero.org/download/).

      Sin embargo, si prefieres gestionar tus aplicaciones de otra manera, puedes instalar la versión en **Flatpak**. Si utilizas Debian o Ubuntu y aún no has configurado Flatpak, sigue estos pasos en la terminal:

      :::tip[¿Utilizas otra distribución?]
      Los siguientes comandos son para distribuciones basadas en Debian/Ubuntu. Si utilizas Fedora, Arch u otra distribución, consulta cómo instalarlo en tu sistema en el [sitio web oficial de Flatpak](https://flatpak.org/setup/) y accede a [Flathub](https://flathub.org/es). Además, conviene comprobar si Zotero está disponible en la tienda de aplicaciones de tu distribución.
      :::

      ```bash
      # 1. Instala la compatibilidad con Flatpak en tu sistema
      sudo apt install flatpak

      # 2. Añade Flathub (el principal repositorio de aplicaciones Flatpak)
      flatpak remote-add --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo

      # 3. Instala Zotero
      flatpak install flathub org.zotero.Zotero
      ```
    </TabItem>
 </Tabs>

#### Extensión del navegador (Zotero Connector)

También debes instalar la extensión del navegador para continuar con el curso. Esta extensión te ayudará a enviar artículos, páginas web, etc., a Zotero.

<Tabs>
  <TabItem label="Chrome / Edge / Brave / Firefox" icon="window">
    Accede a la [página oficial de descarga](https://www.zotero.org/download/). A la derecha, junto a la descarga de la aplicación, encontrarás la opción para instalar Zotero Connector.
  </TabItem>
  <TabItem label="Safari" icon="apple">
    Zotero Connector para Safari ya viene integrado en la aplicación principal de Zotero para macOS.
    Para activarlo, abre Safari, ve a **Safari > Configuración > Extensiones** y marca la casilla de Zotero.
  </TabItem>
</Tabs>

## Configuración

Con la aplicación y la extensión instaladas, vamos a configurar Zotero:

### 1. Ajustar el idioma

Si la aplicación no está en el idioma deseado, sigue estos pasos:

<Steps>
1. Abre Zotero y accede a las preferencias:
   * **En Windows/Linux:** Haz clic en el menú `Editar` (Edit) > `Ajustes` (Settings).
   * **En macOS:** Haz clic en el menú `Zotero` (junto a la manzana) > `Ajustes` (Settings).
2. Ve a la pestaña **General** (General).
3. En la sección Idioma (Language), selecciona **Español**.
4. Haz clic en **Aceptar** y reinicia la aplicación para aplicar los cambios.
</Steps>

---

### 2. Añadir el estilo ABNT

Zotero ya incluye varios formatos internacionales (como APA y Vancouver), pero necesitamos añadir las normas de la ABNT para nuestros trabajos.

<Steps>
1. Vuelve al menú de **Preferencias** y haz clic en la pestaña **Citar** (Cite).
2. Haz clic en el enlace **Obtener estilos adicionales...** (Get additional styles...).
3. Se abrirá una ventana del Repositorio de estilos. En la barra de búsqueda, escribe `ABNT` o `Associação Brasileira de Normas Técnicas`.
4. Haz clic en el estilo deseado para instalarlo automáticamente en tu lista.
</Steps>
